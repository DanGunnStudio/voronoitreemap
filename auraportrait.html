About:

Trying to create a program to replicate the “aura portrait” from “Reading the Auras of Democrats and Republicans on the Eve of the Election” in the NY Times. Specifically this is intended to represent sentiment data from NLP.

Storing Label names and locations for la

label_positions <- purrr::map_dfr(names(tm_aura@cells), function(cell_name) {
  cell_data <- tm_aura@cells[[cell_name]]
  tibble::tibble(
    label = cell_data$name,
    x = cell_data$site[1],
    y = cell_data$site[2]
  )
})
